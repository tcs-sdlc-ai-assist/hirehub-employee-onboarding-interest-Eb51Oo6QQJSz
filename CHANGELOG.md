# Changelog

## [1.0.0] - 2024-06-XX

### Added

- Initial setup with Vite, React 18+, TypeScript, and React Router v6.
- Submission management: view, edit, and track candidate submissions.
- Department management: view department details and assign candidates.
- Form validation and error handling for onboarding submissions.
- Typed interfaces for Submission, Department, FormErrors, and EditFormData.
- Modular code structure for easy maintenance and scalability.

**Note:** This is the first release of the HireHub Onboarding Portal.
